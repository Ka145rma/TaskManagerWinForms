# TaskManagerWinForms

Современное desktop-приложение для управления задачами, разработанное на платформе Windows Forms с использованием C# и .NET. Проект предоставляет удобный интерфейс для создания, редактирования и отслеживания задач в режиме реального времени.

## Возможности

- Добавление новых задач
- Редактирование существующих записей
- Удаление задач
- Изменение статуса выполнения
- Удобная сортировка и отображение данных
- Интуитивно понятный интерфейс Windows Forms
- Работа с локальным хранением данных

---

## Используемые технологии

- C#
- .NET Framework / .NET
- Windows Forms
- Visual Studio

---

## Интерфейс приложения

Приложение включает в себя:

- главное окно со списком задач;
- форму добавления и изменения задач;
- элементы фильтрации и управления;
- отображение текущего статуса задач.
